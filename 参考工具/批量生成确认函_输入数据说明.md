# 📋 华研-夏宝 技术确认函自动生成插件 输入数据说明书

本插件（`batch_generator_all.py`）通过解析程序映射、送样参数数据库以及全局 Excel 模版，全自动生成符合夏宝规范的技术确认函。为了保证插件正常运行并生成高精度、零空行的确认函，以下是插件所需的输入内容、文件结构及字段规范说明。

---

## 📂 1. 输入文件结构概览

插件运行的根目录下需要准备以下几类文件：

```text
📂 批量写入确认函/
 ├── 📄 1.04.01.03047CF_确认函.xlsx        # 全局 Excel 模版文件
 ├── 📄 {系列}程序名称.xlsx                 # [程序映射表]（如：2613程序名称.xlsx）
 ├── 📄 {系列}送样信息副本.xlsx             # [送样参数表]（如：2613送样信息副本.xlsx）
 └── 📄 batch_generator_all.py             # 插件核心脚本
```

---

## 📄 2. 详细输入文件规范

### 2.1 全局 Excel 模版 (`1.04.01.03047CF_确认函.xlsx`)
* **作用**：提供统一的技术确认函样式设计、字号字体、以及单元格合并规则。
* **修改基准**：插件会自动识别并在指定单元格覆写新值，同时清除多余的空行，确保排版紧凑。

---

### 2.2 程序名称映射表 (`{系列}程序名称.xlsx`)
* **作用**：提供需要批量生成的所有物料编码，并映射其对应的固件 `.hex` 程序文件。
* **表格结构规范**：
  * 数据从 **第 2 行** 开始。
  * 包含以下三列（列顺序不可随意调换）：

| 列字母 | 字段名称 | 示例数据 | 插件作用及提取逻辑 |
| :---: | :--- | :--- | :--- |
| **A** | **华研编码** | `1.01.01.01291` | 主索引 Key（用来关联送样参数） |
| **B** | **夏宝编码** | `1.04.01.03009AD` | 次索引 Key，同时作为输出 Excel 的文件名（如：`1.04.01.03009AD_确认函.xlsx`） |
| **C** | **程序名称 (Hex)** | `1.01.01.01291_HYBP2613GK-35RD-01_V1.0_0x696D_260603.hex` | 1. 写入 `B13` 程序名称栏。<br>2. 提取最后的 `_V1.0_0x696D_260603` 部分作为软件版本（`V1.0`）、校验码（`0x696D`）和盖章日期（格式化为 `2026.06.03`，写入 `I16`）。<br>3. 从板子型号（如 `HYBP2613GK-35RD-01`）后缀提取容量代码（`35RD` -> `35`）用作国内外销的自适应判定。 |

---

### 2.3 送样信息副本表 (`{系列}送样信息副本.xlsx`)
* **作用**：提供该系列各编码在研发送样时锁定的各项硬件与电机参数。
* **表格结构规范**：
  * **第一列（A）**：华研编码或夏宝编码（用于和映射表关联）。
  * **第三列（C）** 或 **第四列（D）**：**送样长文本参数列**（插件会自适应搜索包含有 `+` 分隔符或 `制冷` 字样的参数列）。

#### 📝 送样参数长文本（C列）特征词解析规则：
插件注册了多个解析通道（Pipeline Extractors），会自动在长文本中提取以下特征，转化为 Excel 单元格内容：

```text
示例长文本：
"35冷暖变频+标准通讯协议+R32防爆+制冷：1100/1000/900/850 制热：1050/950/850/800/750+测试模式转速1200+wifi模块PID:say9thc7z0gf8x1x-SHARBO客户"
```

1. **容量与国内外销自适应**：
   * 包含大写 `K`（如 `12K`, `9K`）：自动识别为**外销**，**销售地区 (E3)** 设为 `"外销"`，**掉电记忆 (Row 8)** 状态自动设为 `"开启"`，描述填充 `"外销型号默认开启"`。
   * 不含大写 `K`（如 `35`, `26`）：自动识别为**内销**，**销售地区 (E3)** 设为 `"内销"`，**掉电记忆 (Row 8)** 状态自动设为 `"关闭"`，描述填充 `"默认关闭"`。
2. **WiFi 功能提取**：
   * 搜索 `PID:` 或 `pid:`，捕获后续 **16 位字母+数字** 组成 PID，并匹配 `-` 或 `+` 后面的客户名称。写入 Row 8 (如果有WiFi)。
3. **电机转速列表**：
   * 搜索 `制冷：` 和 `制热：` 提取以 `/` 分隔的转速值，自动灌入 `H6` 单元格中。
4. **测试模式转速**：
   * 提取 `测试模式转速` 后的具体描述（如 `测试模式转速1200` 或 `测试模式转速-最小制冷900，其他1200`），填入功能栏中的转速行。
5. **款式提取 (Style)**：
   * 扫描括号内有且仅有一个大写字母的标识（如 `(G款)`、`（Q款）`），大写字母属于款式表时判定为款式，并与容量拼接作为内机箱体（`B5`）内容（例如 `G款35变频挂机`）。

## 💻 3. 脚本代码级配置参数

在 `batch_generator_all.py` 的 `main()` 入口函数中，每个芯片系列的生成规则均通过 `process_chip_series()` 函数进行控制。你可以根据每次的硬件要求在脚本底部修改这些配置：

```python
# 示例配置：调用 process_chip_series
succ_2613, fail_2613 = process_chip_series(
    series_name="2613",                                      # 系列名称（对应生成的目标文件夹名称）
    prog_file="2613程序名称.xlsx",                             # 映射表文件名
    drv_file="2613送样信息副本.xlsx",                         # 送样参数表文件名
    hardware_ver="HYBP2613GK [V1.0]",                        # [写入 B6] 内机硬件型号
    signature_name="何章彬",                                  # [写入 I16] 签署人的中文姓名
    pipeline=pipeline,                                       # 解析管道实例
    writer=writer,                                           # 模板写入器
    base_out_dir=base_output_dir,                            # 输出的根目录（批量生成确认函/）
    mcu_model="美仁MR88FP05DI/032PA_LQFP32",                 # [写入 B13] 对应的 MCU 规格型号
    confirm_type="",                                         # [写入 B13] 确认级别前缀（可选 "首次确认。"、"再次确认。" 或 ""）
    change_reason="对应HYBP2527GK电脑板,PG电机控制由固态继电器方案改为贴片可控硅方案", # [写入 B13] 变更说明原因
    motor_type_override="PG电机"                              # [写入 F5] 强制覆写的内电机类型（如 "PG电机" 或 "直流"）
)
```

## 🛠️ 4. 运行及核验指南

### 运行方式
准备好输入文件并修改好配置后，在安装有 `openpyxl` 的 Python 环境下，通过命令行运行：
```powershell
py batch_generator_all.py
```

### 自动排版核验（生成结果）
1. **自动消空空行**：插件会根据是否有 WiFi 功能，自动在主要功能表（Row 8 - Row 12）顺序渲染 `WiFi功能`、`掉电记忆`、`转速`、`附加功能` 和 `遥控器`，多余的行置空收缩，保持行间距无缝。
2. **防OneDrive锁定**：插件内置了文件夹自动补全和 OneDrive 并发占用防御策略，如果生成中发现文件夹不存在会即时创建，确保写入 100% 成功。
